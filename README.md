### 一、相关技术

- 前端：`Vue`、`HTML` / `CSS` / `Javascript`。
- 后端：`Springboot`等。
- 数据库：`MySQL`

### 二、开发软件（列出的软件其一均可运行）

- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`

### 三、部署所需环境

- 后端：`jdk1.8`。
- 前端：`node`、`vue-cli` 、`npm`  。
- 数据库：`mysql5.7以上`

### 四、功能描述及功能图展示

系统分为：管理员。

管理员功能：1. 登录、2. 集装箱信息管理、3. 进出库信息管理、4. 堆场位置信息管理、5. 人员信息管理、6. 设备调度管理、7. 装卸信息管理、8. 作业进度跟踪、9. 作业质量监控
